## Hugeicons Core Free Icons

This project includes adapted Android Vector Drawable versions of the following icons from `@hugeicons/core-free-icons` 4.2.2:

- `Settings01Icon`
- `ArrowUp01Icon`
- `ArrowDown01Icon`
- `ArrowLeft01Icon`
- `ArrowRight01Icon`
- `KeyboardIcon`
- `SentIcon`
- `Globe02Icon`

The icon geometry, 24 × 24 grid, rounded strokes, and 1.5 stroke width follow the official Stroke Rounded source. Only the Android resource format and theme-driven color were changed.
